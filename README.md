# A list of different game tools and what they offer
<br />
<br />
<br />
<br />
<br />
<br />
<br />
<br />

## 1. LINGO <img src="https://github.com/user-attachments/assets/044b85fa-4db9-4d5c-9627-8f17aad69b9c" width="300" height="300">
Modeling Software for Linear, Nonlinear, and Integer Programming. This software claims to be good at optimizing models. Released in 1989. It was used to make the prototype for the indie game RainWorld.

## 2. Futile Framework, by MattRix <img src="https://github.com/user-attachments/assets/4ed88c7e-6674-4fcb-9608-f35a2a7ab8f2" width="300" height="300">
A code-centric 2D framework for Unity. Allows for less component work, and more code to be used in Unity. Works a lot like Cocos2D and Flash. It was and is used in many games like RainWorld.

## 3. Flash <img src="https://github.com/user-attachments/assets/dcedbcbf-ed35-49b3-a52e-0cc3696aba3f" width="300" height="300">
Released in 1996. This engine was used for making animations, advertisements, and many web embedded games. Projects in this software are programmed with the language Flash. One game that was built on Flash was Super Hexagon.

## 4. Source <img src="https://github.com/user-attachments/assets/8324efa7-5196-493b-9a61-c3f8a4dc971e" width="700" height="200">
Source was a game engine developed by Valve Corporation in 2004. It was used to make popular games like Counter Strike, Half-Life 2, and Portal.

## 5. Cocos2D <img src="https://github.com/user-attachments/assets/e16463f6-6da0-4774-906e-293e33b14c69" width="200" height="350">
Cocos2D was a game framework built in C++, and can be used to bind multiple languages such as C++, C#, Lua, and JavaScript. It was made in 2008 and originally coded in Python. One popular game that was built with Cocos2D was Geometry Dash.

## 6. Greenfoot <img src="https://github.com/user-attachments/assets/91f43627-8aa6-4f87-98cd-17856ced8606" width="200" height="250">
Greenfoot is a game engine published in 2006. Languages that can be used are Java and Stride. This software is mainly used to teach people to interactively code in Java.

## 7. Raylib <img src="https://github.com/user-attachments/assets/26de826a-cc6b-4003-b09d-a6670e5ea5d0" width="250" height="250">
Raylib is a highly modular development library built in C and can bind over 60 languages. It was released in 2013, and aims to be a barebones piece of software to make games from nearly the ground up.

## 8. Delta Engine <img src="https://github.com/user-attachments/assets/88687607-8ba4-4dfa-85ae-58bfd97cea59" width="200" height="200">
Delta Engine was created in the 1990s. It created the first commercial real time strategy game built on the .NET framework in early 2000s. Delta Engine can be developed in C++ and Python. Delta Engine's most popular game is SoulCraft.

## 9. Babylon.js <img src="https://github.com/user-attachments/assets/f0215eab-f8ee-4d7e-b686-e9cc9cfcfee3" width="225" height="200">
Babylon.js is a web rendering engine made with WebXR to develop programs on the web. It was released in 2013 and can be developed in C# and Typescript. One game made inn Babylon.js is Minecraft Classic.

## 10. Ceramic Engine <img src="https://github.com/user-attachments/assets/bdd49a02-e41c-4f93-b258-ee57f6b7a491" width="350" height="200">
Ceramic Engine is an open-source 2D framework released in 2020. Developers have to write their code in the Haxe language and the engine will transpile it to C++, JavaScript, or C#.

## 11. Unity <img src="https://github.com/user-attachments/assets/41b5060c-9f68-4c7a-b719-0a420c0a61f8" width="450" height="250">
Unity is a game engine packed with their own framework. Unity was released in 2005. Developers are limited to C# or GUI only programming. Many popular games were built on Unity like RainWorld, Universe Sandbox 2, and City Skylines 2.

##12. Godot <img src="https://github.com/user-attachments/assets/2be9f1c8-e941-475e-8ad8-70a1aa3639b9" width="300" height="300">
Godot is a free and open source game engine. The software was released in 2014. Godot can be developed in GDScript, C/C++, and C#. Some games made in Godot are Dome Keeper and Brotato.

## 13. Unreal Engine <img src="https://github.com/user-attachments/assets/a85a48c8-dace-4275-bb05-ef94268597b7" width="300" height="300">
The original version of Unreal Engine was released in 1998 for a FPS game called Unreal. It has since gone through many iterations, and is now mostly agreed on as the most powerful and professional-grade game engine. Unreal Engine 5 can be developed in both C++ and their own node-based interface, Unreal Blueprint. Popular games made in Unreal Engine are Fortnite, Tetris Effect, Borderlands 3, Bioshock, PUBG, and Rocket League.
